\# Present Perfect Masterclass 🎓



An interactive, single-file web-based presentation and quiz application designed to teach the English "Present Perfect" tense. Built with modern web technologies focusing on a premium UI/UX, self-study capabilities, and gamified learning.

\## ✨ Key Features

\- \*\*Single File Architecture:\*\* HTML, CSS, and Vanilla JavaScript combined in one file for ultimate portability.

\- \*\*Modern UI/UX:\*\* Glassmorphism design, animated ambient orbs, SVG noise background, and smooth transitions.

\- \*\*Self-Study Optimized:\*\* Clear rules, timelines, and a detailed comparison between \*Present Perfect\* and \*Past Simple\*.

\- \*\*Interactive Quizzes:\*\* "Click-to-fill" gap mechanics with real-time feedback, shake-on-error animations, and success pop-ups.

\- \*\*Two-Tier Assessment:\*\* 

&nbsp; - \*In-class Practice:\* Guided practice with correct answer reveals.

&nbsp; - \*Final Test:\* Scored test required to unlock the certificate.

\- \*\*Certificate Generation:\*\* Dynamic certificate creation with the student's name and score, fully optimized for A4 landscape printing.

\- \*\*Keyboard Navigation:\*\* Support for `ArrowLeft` and `ArrowRight` to navigate through slides.



\## 🚀 How to Run (Local Usage)

Since this project uses zero external dependencies and requires no build tools:

1\. Clone this repository or download the `index.html` file.

2\. Double-click `index.html` to open it in any modern web browser (Chrome, Firefox, Safari, Edge).

3\. Start learning!



\## 🛠️ Technologies Used

\- \*\*HTML5:\*\* Semantic structure and layout.

\- \*\*CSS3:\*\* Custom variables, Flexbox/Grid, CSS Animations, Backdrop-filter (Glassmorphism), and `@media print` for the certificate.

\- \*\*Vanilla JavaScript (ES6+):\*\* Slide navigation logic, array manipulation for quiz generation, state management, and DOM manipulation.



\## 📝 Customization

You can easily adapt this template for other grammar lessons (e.g., Past Simple, Future Perfect):

1\. \*\*Change Content:\*\* Find the `<div class="slide">` tags and update the `<h2>` and `<p>` texts.

2\. \*\*Update Quizzes:\*\* Scroll to the `<script>` section and modify the `classQuestions` and `hwQuestions` arrays. 

&nbsp;  - `t`: Text array (use `"\_"` for blanks).

&nbsp;  - `w`: Word bank options.

&nbsp;  - `a`: Correct answers array.

3\. \*\*Change Branding:\*\* Update the `mrabbas.ir` watermark and author names in the HTML structure.
